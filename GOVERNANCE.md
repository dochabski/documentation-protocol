# Governance

This repository is governed by the source-of-truth model in `specs/product-governance-and-stewardship.yaml`.

## Draft Roles

- Protocol owner: to be assigned before public release.
- Maintainer: to be assigned before public release.
- Reviewer: assigned per protocol product, package, or release candidate.
- Release manager: assigned before tagging or publishing a release.
- Steward: assigned for long-term preservation and continuity.

## Decision Records

Protocol decisions that affect requirements, conformance, templates, schemas, automation, release, or preservation should be recorded in `records/`.

## Publication Blockers

- Identify initial maintainers and governance contact.
- Decide first pilot artifact.
- Decide whether GitHub workflow files are included before or after pilot review.
